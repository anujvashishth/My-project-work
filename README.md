# Dairy Management System (Mini Project)

A modern, fast, and beautiful web-based Dairy Management System built with React and Vite.

## 🚀 Getting Started

### Prerequisites
- Node.js installed on your system.

### Installation
1.  Navigate to the project directory:
    ```bash
    cd dairy-system
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App
1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## 🛠️ Features
- **Admin Dashboard:** Overview of total collections, customers, and financials.
- **Milk Collection:** Simple form to enter quantity, FAT%, and SNF% for farmers.
- **Customer Management:** List of all registered farmers with their details.
- **Reports:** Visualized data and exportable summaries.

## 🎨 Design Philosophy
- **Glassmorphism:** Modern translucent UI with soft blurs.
- **Micro-animations:** Smooth transitions using Framer Motion.
- **Responsive:** Adapts to different screen sizes.

## 📚 Project Structure
- `src/App.jsx`: Main logic and routing between views.
- `src/index.css`: Design system and styling.
- `src/main.jsx`: Entry point for the application.
